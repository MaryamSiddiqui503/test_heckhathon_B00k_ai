## üìò Project Overview

This repository contains a **complete digital textbook** for the course  
**Physical AI & Humanoid Robotics**.

The book focuses on **Physical AI and embodied intelligence**, explaining how
artificial intelligence systems operate in the **physical world** through
humanoid robots. It bridges the gap between digital intelligence and physical
bodies using modern robotics tools and simulations.

The textbook is created using **Spec-Driven Development** with **Spec-Kit Plus**
and authored using **Claude Code**.

### Module 1: The Robotic Nervous System (ROS 2)

- Visual SLAM and navigation

Each module consists of **exactly three chapters**, following a structured,
academic approach.
